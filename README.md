# c-calendar

Calendar program written in C programming language. 

## Why C?

I have chosen to write it in C because that is the first programming language I have learned and I wanted to refresh my memory.

## Why Calendar?

Calendar is a bit more complex then usual input/output beginner program, and it was also my graduation paper - to make calendar in C.

## Why now?

My plan is to write one small C program at least once every two months, to keep my brain cells working. Modern programming languages have made us incredibly lazy with automatic garbage collection, memory allocation, etc. so this is a perfect way tp keep the brain cells running.

The LLMs are also not helping this cause, using them, we can really forget how to think like engineers and solve problems, so this is my way of keeping my brain stamina.

## Notice

No AI was used or will be used in any part of writing this program. I have used only Wikipedia, GeeksForGeeks and W3 Schools websites as my reference. I try to use the minimum amout of libraries as possible, and to do the most by hand and always having the control of the memory.
